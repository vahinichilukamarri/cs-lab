Cyber Security Lab Commands

1. Disk Imaging & Forensics

# STEP 1: Create evidence file
echo "Cybersecurity Lab Evidence" > evidence.txt

# STEP 2: Verify file
ls

# STEP 3: Check disks
lsblk

# STEP 4: Install dc3dd
sudo apt update
sudo apt install dc3dd -y

# STEP 5: Create 100MB practice disk
dd if=/dev/zero of=practice_disk.dd bs=1M count=100

# STEP 6: Format disk
mkfs.ext4 practice_disk.dd

# STEP 7: Create mount directory
sudo mkdir /mnt/practice_disk

# STEP 8: Mount disk
sudo mount -o loop practice_disk.dd /mnt/practice_disk

# STEP 9: Copy evidence into mounted disk
sudo cp evidence.txt /mnt/practice_disk/

# STEP 10: Verify copied file
ls /mnt/practice_disk

# STEP 11: Unmount disk
sudo umount /mnt/practice_disk

# STEP 12: Create forensic image
dc3dd if=practice_disk.dd of=disk_image.dd hash=md5 log=acquisition.log

# STEP 13: Verify image
ls -lh disk_image.dd

# STEP 14: View acquisition log
cat acquisition.log

# STEP 15: Start Autopsy
autopsy
Then open browser inside Kali and go to:

http://localhost:9999/autopsy

Inside Autopsy:

Create New Case
Add Host
Add Image
Select:
/home/kali/disk_image.dd
Open File Analysis
Locate:
evidence.txt
Verify content:
Cybersecurity Lab Evidence
